Operación del Videojuego
Al iniciar, el jugador visualiza una flota de naves espaciales enemigas como Tie Fighters y aliadas como X-Wings que se desplazan y rebotan entre sí. El usuario debe utilizar el puntero para hacer clic sobre los objetivos, al acertar el objeto desaparece con una animación de explosión, se reproduce un efecto de sonido y se actualizan los contadores en el HUD.
Eliminar enemigos otorga puntos positivos, hasta +100 por un Star Destroyer, mientras que impactar naves aliadas penaliza al jugador restando puntos del marcador. El HUD lateral muestra en tiempo real el puntaje total, el número de objetos actuales y el conteo detallado de bajas aliadas y enemigas. Además, el juego cuenta con un panel de reglas desplegable y un botón de reinicio para restablecer todas las estadísticas y objetos a su estado inicial.
